# 🧠 AI Quiz Platform

An AI-powered quiz generator built with **Streamlit and Groq LLM**.  
Generate quizzes on any topic and test your knowledge instantly.

---

## 🚀 Features

- AI-generated quiz questions
- Difficulty levels (Easy, Medium, Hard)
- Quiz timer system
- User login & signup
- Quiz history
- Leaderboard
- Downloadable quiz reports (PDF)
- Quiz completion certificates
- Beautiful UI with custom CSS

---

## 🛠 Tech Stack

- Python
- Streamlit
- Groq API
- SQLite
- Pandas
- FPDF
- HTML/CSS

## ⚙️ Installation

Clone the repository

git clone https://github.com/gungunsingh66/ai-quiz-app.git

Install dependencies

pip install -r requirements.txt

Run the app

streamlit run app.py
